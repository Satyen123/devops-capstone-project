# CI/CD Final Project

This project demonstrates Continuous Integration and Continuous Delivery (CI/CD) using GitHub Actions, Tekton Pipelines, and OpenShift.

## Features

- Automated linting with flake8
- Automated unit testing with nose
- Tekton pipeline tasks
- Build and deploy application to OpenShift
